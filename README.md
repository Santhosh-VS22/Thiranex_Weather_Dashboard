# 🌦 Thiranex Task 4 - Weather Dashboard

## 📌 Project Overview

This project was developed as **Task 4** of the **Thiranex Web Development Internship**.

The Weather Dashboard retrieves real-time weather information from the **Open-Meteo REST API** using JavaScript Fetch API and Async/Await.

Users can search for any city and instantly view live weather details including temperature, humidity, wind speed, weather condition, and country.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡 Live temperature
- 💧 Humidity
- 💨 Wind speed
- 🌤 Weather condition
- 🌍 Country information
- ⚡ Fetch API
- ⏳ Async / Await
- 📦 REST API Integration
- 📄 JSON Parsing
- ❌ Error Handling
- 📱 Fully Responsive Design
